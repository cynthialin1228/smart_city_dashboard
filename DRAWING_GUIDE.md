# 🎯 ROI 和計數線繪製指南

## 理想設定

假設影片是標準十字路口，以下是完美的繪製方式：

### 📐 ROI（興趣區域）

**目標**: 覆蓋整個十字路口，涵蓋 4 條計數線

**推薦座標** (歸一化 0.0~1.0):
```
ROI 多邊形的 4 個頂點：
  左上: (0.15, 0.15)
  右上: (0.85, 0.15)
  右下: (0.85, 0.85)
  左下: (0.15, 0.85)
```

**繪製步驟**:
1. 點擊 "ROI Mask" 按鈕 (綠色)
2. 在影片上依序點 4 下：
   - 左上角
   - 右上角
   - 右下角
   - 左下角
3. 最後在第 1 個點附近點一下或雙擊來閉合
4. 看到半透明藍色區域表示成功

## 🚦 計數線（A, B, C, D）

### 線段佈局

標準十字路口配置：

```
        ↑ D (垂直)
        |
    ←─────────→ A (水平)
    B   ●   C
        |
        ↓ D (垂直)
```

### 理想座標

#### **A 線** 🟢 綠色 (直行 →)
- **方向**: 完全水平，從左到右
- **高度**: 中間偏上 (y ≈ 0.4)
- **座標**:
  ```
  起點: (0.0, 0.4)
  終點: (1.0, 0.4)
  ```
- **用途**: 計算向右行駛的車輛

#### **B 線** 🟠 橙色 (對向 ←)
- **方向**: 完全水平，從右到左（與 A 平行）
- **高度**: 中間偏下 (y ≈ 0.6)
- **座標**:
  ```
  起點: (1.0, 0.6)
  終點: (0.0, 0.6)
  ```
- **用途**: 計算向左行駛的車輛

#### **C 線** 🟣 紫色 (左轉/下行 ↓)
- **方向**: 完全垂直，從上到下
- **位置**: 左側 (x ≈ 0.3)
- **座標**:
  ```
  起點: (0.3, 0.0)
  終點: (0.3, 1.0)
  ```
- **用途**: 計算向下行駛的車輛

#### **D 線** 🟡 黃色 (右轉/上行 ↑)
- **方向**: 完全垂直，從下到上（與 C 平行）
- **位置**: 右側 (x ≈ 0.7)
- **座標**:
  ```
  起點: (0.7, 1.0)
  終點: (0.7, 0.0)
  ```
- **用途**: 計算向上行駛的車輛

## ✅ 繪製清單

### Step 1: 繪製 ROI

- [ ] 點擊綠色 "ROI Mask" 按鈕
- [ ] 在影片上點 4 次形成多邊形
- [ ] 看到半透明藍色區域
- [ ] ROI 應涵蓋整個十字路口

### Step 2: 繪製 A 線

- [ ] 點擊 "A" 按鈕（自動變綠色）
- [ ] 在影片**左邊緣**點一下（起點）
- [ ] 在影片**右邊緣**同高度點一下（終點）
- [ ] 看到綠色水平線條
- [ ] 自動跳到 B

### Step 3: 繪製 B 線

- [ ] 點擊 "B" 按鈕（自動變橙色）
- [ ] 在影片**右邊緣**點一下（起點）
- [ ] 在影片**左邊緣**同高度（但低於 A）點一下（終點）
- [ ] 看到橙色水平線條
- [ ] 自動跳到 C

### Step 4: 繪製 C 線

- [ ] 點擊 "C" 按鈕（自動變紫色）
- [ ] 在影片**上邊緣**點一下（起點）
- [ ] 在影片**下邊緣**同水平位置點一下（終點）
- [ ] 看到紫色垂直線條
- [ ] 自動跳到 D

### Step 5: 繪製 D 線

- [ ] 點擊 "D" 按鈕（自動變黃色）
- [ ] 在影片**下邊緣**點一下（起點）
- [ ] 在影片**上邊緣**同水平位置（但右於 C）點一下（終點）
- [ ] 看到黃色垂直線條

### Step 6: 確認發送

- [ ] 所有 4 條線都已繪製完成
- [ ] "🚀 LAUNCH AI ANALYSIS" 按鈕應亮起
- [ ] 點擊 LAUNCH 進入戰情室

## 🎨 視覺參考

### 正確範例

```
0.0                           1.0
|                             |
+─────────────────────────────+  0.0
|         ROI 藍色區域         |
|   +─────────── A 線 ────────+  0.4
|   |                        |
|   |      十字路口          |
|   |      (標誌點)          |
|   +─────────── B 線 ────────+  0.6
|   |                        |
|   C  |                  |  D
|   線  |      車流        |  線
|   |  |                  |  |
|   |  |   ●    ●    ●   |  |
|   |  |   ●    ●    ●   |  |
|   v  v                  v  ^
+─────────────────────────────+  1.0
```

### 常見錯誤 ❌

```
❌ 線段太短（只在角落）
  A 線: (0.3, 0.4) → (0.7, 0.4)  ← 應該到邊界
  修正: (0.0, 0.4) → (1.0, 0.4)  ✅

❌ 線段方向錯誤
  C 線: (0.3, 0.3) → (0.3, 0.7)  ← 太短
  修正: (0.3, 0.0) → (0.3, 1.0)  ✅

❌ 平行線距離太近
  A 線: (0.0, 0.45) → (1.0, 0.45)
  B 線: (1.0, 0.50) → (0.0, 0.50)  ← 太接近
  修正: (1.0, 0.6) → (0.0, 0.6)   ✅
```

## 📊 實例

### 典型十字路口

```
假設影片是 1920×1080 的十字路口：

ROI 頂點（歸一化後）:
  (0.15, 0.15) → (288, 162)      左上
  (0.85, 0.15) → (1632, 162)     右上
  (0.85, 0.85) → (1632, 918)     右下
  (0.15, 0.85) → (288, 918)      左下

計數線:
  A: (0, 432) → (1920, 432)       y = 0.4
  B: (1920, 648) → (0, 648)       y = 0.6
  C: (576, 0) → (576, 1080)       x = 0.3
  D: (1344, 1080) → (1344, 0)     x = 0.7
```
