# 智慧城市在貿易廊道的應用

- Objective: 真實道路影片資料，由 AI 偵測車流狀況
- 對象：交通局 / 城市交通管理單位
- Input: 實際道路監視器影片，有車流影像資料，包含直行車、對象來車、左轉車、橫向路口車
- Output：一個dashboard（可視化儀表板），形式自由，內容包含車輛偵測、車種分類、車流計數、時段統計、可視化儀表板
- 角色

## 任務流程參考 Pipeline
建立從影片資料到交通決策輔助的參考流程，但可以自己調整

1. 資料洞察與問題定義: 提供道路影片，分析相機角度、畫質、車流方向、道路型態與計數線位置。
2. AI 車輛辨識與車流計數模型: 針對汽車、機車、巴士等車種進行辨識與追蹤，計算通過指定線段的車流量。(maybe use YOLO for object detection)
3. **重點** 決策建議轉換：根據車流量等等資料提供具體建議。
4. 具體建議產出：以單一 localhost 頁面 Dashboard 呈現關鍵車流指標，比如車流量、車種比例、方向流量與壅塞熱點，雍塞指標等協助政府單位快速掌握道路狀況，並支援交通管理與改善決策。
