# ⚡ 快速開始 (5 分鐘)

## 🎯 理想 ROI 和線段座標

### ROI（興趣區域）
```
左上: (0.15, 0.15)
右上: (0.85, 0.15)
右下: (0.85, 0.85)
左下: (0.15, 0.85)
```

### 計數線 (A, B, C, D)

| 線 | 顏色 | 座標 | 方向 |
|----|------|------|------|
| **A** | 🟢 綠 | `(0.0, 0.4)` → `(1.0, 0.4)` | 水平 → |
| **B** | 🟠 橙 | `(1.0, 0.6)` → `(0.0, 0.6)` | 水平 ← |
| **C** | 🟣 紫 | `(0.3, 0.0)` → `(0.3, 1.0)` | 垂直 ↓ |
| **D** | 🟡 黃 | `(0.7, 1.0)` → `(0.7, 0.0)` | 垂直 ↑ |

```bash

cd backend

cd frontend
npm run dev

# 打開瀏覽器
http://localhost:5173
```

## 📝 繪製步驟

1. **ROI**: 點 4 個頂點圍成藍色區域（覆蓋整個十字路口）
2. **A 線**: 完全水平，從左邊緣到右邊緣（上方）
3. **B 線**: 完全水平，從右邊緣到左邊緣（下方）
4. **C 線**: 完全垂直，從上邊緣到下邊緣（左側）
5. **D 線**: 完全垂直，從下邊緣到上邊緣（右側）
6. **LAUNCH**: 點擊 🚀 按鈕進入戰情室

## 🐛 常見問題

| 症狀 | 原因 | 解決 |
|------|------|------|
| PCU 全 0 | 線段無效 | 重新畫線，確保完整穿過影片 |
| 粒子不動 | 連線失敗 | 檢查後端是否啟動，查 TROUBLESHOOTING.md |
| Canvas 不對齎 | 尺寸計算錯 | 打開 `?debug=1` 面板檢查 |

更多問題 → **[TROUBLESHOOTING.md](./TROUBLESHOOTING.md)**
